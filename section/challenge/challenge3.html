Challenge 3: BMI Calculator Design

BY: CHALLENGE MASTER TAKAKU

In this challenge, you'll be tasked with designing a simple web interface for a BMI calculator using CSS. The goal is to bring life to the BMI calculator created with HTML and JavaScript by adding creative and functional designs.

Challenge Instructions

Objective: Use CSS to design a visually appealing and functional BMI calculator interface. The calculator is already built using HTML and JavaScript, and your task is to enhance its appearance using CSS.

Steps:

  1. Set up your development environment with a compiler that supports HTML, CSS, and JavaScript. We recommend using Visual Studio Code.
  2. Install all necessary extensions to run HTML, CSS, and JavaScript files.
  3. Create three files:
    • Name the HTML file as start.html.
    • Name the JavaScript file as start.js.
    • Name the CSS file as start.css.
  4. Copy the provided HTML and JavaScript code into their respective files.
  5. Use the CSS file to design the BMI calculator. Be creative with colors, layouts, and animations!
  6. Take a screenshot of your CSS code and the final product, then share it in the #coding-challenges channel.

Provided Code:

Below is the HTML and JavaScript code for the BMI calculator. Paste these into their respective files:

HTML:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <link rel="stylesheet" href="start.css">
  <title>BMI Calculator</title>
</head>
<body>
  <div class="container">
    <h1>BMI Calculator</h1>
    <label for="hinp">Height (m):</label>
    <input type="number" id="hinp" step="0.01">
    <label for="winp">Weight (kg):</label>
    <input type="number" id="winp" step="0.01">
    <button onclick="calculate()">Calculate BMI</button>
    <p id="result"></p>
  </div>
  <script src="start.js"></script>
</body>
</html>

JavaScript:

// JavaScript
function calculate() {
  let height = parseFloat(document.getElementById('hinp').value);
  let weight = parseFloat(document.getElementById('winp').value);

  let msqr = height * height;
  let result = weight / msqr;

  let bmicat;
  if (result < 18.5) { bmicat = 'Underweight'; }
  else if (result < 24.9) { bmicat = 'Normal'; }
  else if (result < 29.9) { bmicat = 'Overweight'; }
  else { bmicat = 'Obese'; }

  document.getElementById('result').innerText = `Your BMI is: ${result.toFixed(2)} (${bmicat})`;
}

Challenge 2: Even or Odd

BY: FOUNDER MATT

In this challenge, you’ll create a simple program that checks whether a number is even or odd. It’s a great way to practice basic conditionals and user input handling.

Challenge Instructions

Objective: Write a program that:

  • Asks the user to input an integer.
  • Checks if the number is even or odd.
  • Prints "Even" if the number is even, and "Odd" if the number is odd.

Steps:

  1. Take an integer input from the user.
  2. Use the modulus operator (%) to determine whether the number is divisible by 2.
  3. If the remainder is 0, the number is even; otherwise, it’s odd.

Expected Output:

If the user inputs 4, the program should output: 

Even

If the user inputs 7, the program should output: 

Odd

Submission:

Post your solution in the #coding-challenges channel with a screenshot of your output. Let’s see who can solve this with the cleanest and most efficient code!

Challenge 1: Hash Triangle

BY:FOUNDER MATT

In this challenge, you'll be tasked with writing a program that generates a right-aligned triangle made of hash symbols (#). The number of rows in the triangle will be determined by the user input.

Challenge Instructions

Objective: Use any Programming Language. Create a program that takes an integer input from the user and prints a right-aligned triangle of hash symbols (#). The triangle should start with the number of # symbols equal to the input value and decrease by one in each subsequent row.

Steps:

  1. Take an integer input from the user. (If the user does not input a positive integer, print an error message.)
  2. Use two loops to print rows of # symbols. The number of # symbols should decrease as the row number increases.
  3. The triangle should be right-aligned, meaning the longest row of # symbols is at the top, and the number of symbols decreases as you move down.

Expected Output:

If the user inputs 5, the output will look like this: 

#####
####
###
##
#

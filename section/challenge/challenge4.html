Challenge 4: Fibonacci Sequence Generator

FIRST CHALLENGE FOR THIS OCTOBER

In this challenge, you'll write a program that generates the Fibonacci sequence up to a given number. The Fibonacci sequence is a series of numbers where each number is the sum of the two preceding ones.

Challenge Instructions

Objective: Use any programming language to create a program that takes an integer input and prints the Fibonacci sequence up to that number.

Steps:

  1. Take a positive integer input from the user.
  2. Generate the Fibonacci sequence (each number is the sum of the previous two) until the next number would exceed the input value.
  3. Print the resulting sequence.

Expected Output:

If the user inputs 20, the output should look like this: 

0, 1, 1, 2, 3, 5, 8, 13

If the user inputs 5, the output will be: 

0, 1, 1, 2, 3, 5

Submission:

Post your solution along with screenshots of your output in the #coding-challenges channel. Feel free to discuss different methods for generating Fibonacci sequences, such as recursion or iteration.
